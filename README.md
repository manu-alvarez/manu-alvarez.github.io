## 📈 Proyectos Destacados

### 🧠 [Ecosistema MSBrossAI](https://msbross.me)
Diseño, desarrollo y orquestación de una suite centralizada de **22 aplicaciones avanzadas** impulsadas por Inteligencia Artificial.
* **Arquitectura:** Proxy reverso global con túneles de Cloudflare, sirviendo múltiples SPAs (Vite, Next.js) e interfaces backend (FastAPI, Express) orquestadas a través de PM2 en un entorno Unix con zero-downtime.
* **Módulos Clave:**
  * **Traductor PRO:** Traductor neural multimodal con soporte OCR (Tesseract.js) y fallback secuencial (Groq, Gemini, OpenAI).
  * **LIVEKIT Nikolina:** Asistente de voz en tiempo real usando WebRTC, Gemini 2.5 Flash Audio y FastAPI.
  * **IndustrialPro:** Sistema SCADA/MES web para control de operaciones, válvulas y telemetría industrial en tiempo real.
  * **TaskFlow Pro:** PWA de productividad con persistencia nativa, Service Workers e integraciones de mensajería (WhatsApp).

### 📱 [App en Play Store - Mano Eléctrica Azul](https://play.google.com/store/apps/details?id=com.manoelectricaazul.app)
Desarrollo de una aplicación multiplataforma (Android) utilizando **React Native y Expo** para la exploración de calendarios mayas y contenido interactivo, gestionando el ciclo completo desde la UI hasta el despliegue en la tienda.

### 📦 [Logitrack - Gestión de Operaciones](https://trello.com/b/IRVpzuUt/logitrack-operaciones-de-almacen)
Implementación de un sistema de gestión visual logístico utilizando tableros Kanban bajo metodologías Scrumban y APIs automatizadas para la optimización de almacenes y rutas de campo.
